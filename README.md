### 🔍 About
JotPad is designed to be more than just a basic text editor. It features a robust categorization system where users can filter notes by folder, track the number of notes in each category, and perform quick actions like editing or deleting notes. 

**Key Features:**
* **Category Management:** Organize notes into dedicated folders with live count updates.
* **Firebase Integration:** Real-time data synchronization and secure storage for your notes.
* **Secure Authentication:** User login and registration system to keep personal data private.
* **Dynamic UI:** Responsive layouts built with modern design principles, including empty state handling and smooth transitions.

---

### 🛠️ Technologies
This project utilizes a modern Android tech stack to ensure high performance and maintainable code:

* **Jetpack ViewBinding** - For safe and efficient interaction with UI components.
* **Coroutines** - To handle asynchronous tasks and database operations without blocking the main thread.
* **Firebase (Auth & Firestore)** - Powers the backend for user management and real-time cloud data storage.
* **ViewModel & LiveData** - Implements architectural patterns to ensure data survives configuration changes.
* **Clean Architecture** - Structured into distinct layers for better scalability and testability.
* **Navigation Component** - Handles fragment transitions and back-stack management.
